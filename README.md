<h2>O que é?</h2>
<p>Bot de discord em Python que requisita as informações da API pública do FlameMC (api.flamemc.com.br) e exibe elas em forma de embed.</p>

<h2>Comandos do Bot</h3>

  <h3>Geral:</h3>
 <p>- /info - Verifica as informações de um jogador</p>
 <p>- /status - Verifica se o servidor está online</p>
 <p>- /convite - Link para convidar o bot para um servidor</p>
  <h3>Leaderboards:</h3>
  <p>- /arena (semanal/mensal/total) - Exibe a leaderboard do ArenaPvP</p>
  <p>- /cxc (semanal/mensal/total) - Exibe a leaderboard do Clan x Clan</p>
  <p>- /flameleague (semanal/mensal/total) - Exibe a leaderboard da FPS</p>
  <p>- /fps (semanal/mensal/total) - Exibe a leaderboard da FPS</p>
  <p>- /hg (semanal/mensal/total) - Exibe a leaderboard do HG</p>
  <h3>Moderação:<h3>
  <p>- /canais adicionar - Adiciona um canal onde é permitido usar comandos</p>
  <p>- /canais remover - Remove um canal onde é permitido usar comandos</p>
  <p>- /canais lista - Lista os canais onde é possivel usar comandos</p>
